### 🏠 Home Server Architecture
> **Self-hosted personal cloud solution**

🔹 Architected and maintained a secure, self-hosted personal cloud for file storage and management  
🔹 Deployed services on **Unraid server** using **Docker containers**  
🔹 Implemented **NextCloud** as the core cloud platform  
🔹 Set up **NGINX with SWAG** as reverse proxy for HTTPS access

🔹 Automated regular backups and resource monitoring for optimal uptime and performance  

**Tech Stack:** `Unraid Server` • `Docker` • `NGINX` • `SWAG` • `MariaDB` • `Cloudflare` • `NextCloud`
